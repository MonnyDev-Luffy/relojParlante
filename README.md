# Reloj Parlante

**Reloj Parlante** es una aplicación que **te dice la hora con voz**, sincronizada con el reloj de tu dispositivo.  
Cada vez que presionas el botón, la app **anuncia la hora exacta** con expresiones naturales como:  
> “Son las tres de la tarde con quince minutos” o “Son las doce de la noche en punto”.

---

## Generalidades

1. Abre la app.
2. La pantalla muestra la **zona horaria, fecha y hora actual**.
3. **Sube el volumen.**  
4. **Presiona el botón “Decir la hora”.**  
5. **Respuesta inmediata:** La app **reproduce la hora con voz**, indicando:
   - “de la mañana” (12 AM – 11:59 AM)
   - “de la tarde” (12 PM – 6:59 PM)
   - “de la noche” (7 PM – 11:59 PM)

---

## Tecnologías

La aplicación está desarrollada en **Kotlin**, utilizando **Android Studio**, **Jetpack Compose**, **MediaPlayer** y **Tiempo real** con `Calendar` y `SimpleDateFormat` sincronizados con el reloj del sistema.
